# mcav-rs

Prints a Minecraft avatar from a skin in the terminal.

## Demo

<img src="https://raw.githubusercontent.com/Zielin0/mcav/master/demo.png" alt="mcav-rs-demo" width="500" />

## Quick Start

Install:
```sh
$ cargo install mcav
```

Use:
```sh
# Normal scale
$ mcav <username>
# Custom scale
$ mcav <username> 2
```
