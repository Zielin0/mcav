# mcav-rs

Prints a Minecraft avatar from a skin in the terminal.

## Demo

<img src="./demo.png" alt="mcav-rs-demo" width="500" />

## Quick Start

Build:
```sh
$ cargo build --release
```

Use:
```sh
# Normal scale
$ target/release/mcav assets/skin.png
# Custom scale
$ target/release/mcav assets/skin.png 2
```
